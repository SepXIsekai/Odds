1 . Clone Repo ลงใน Ubantu<br>
```sh
git clone https://github.com/SepXIsekai/Odds.git
```
2 . Change Directory ไปยังที่ Clone Project ไป<br>
<br>
3 . ติดตั้ง bundle<br>
```sh
bundle install<br>
```
4 . ตั้งค่าฐานข้อมูล<br>
```sh
rails db:create
rails db:migrate
rails db:seed (ถ้ามีข้อมูลเริ่มต้น)
```
5 . Run server<br>
```sh
rails server
```
